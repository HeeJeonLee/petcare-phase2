### 4️⃣ 교육 센터 (InsuranceEducation.jsx) - 450줄
```
📖 보험 용어 8개:
1. 보험료 (Premium) - 월 납부금
2. 보장금액 (Coverage) - 최대 보장액
3. 자기부담금 (Deductible) - 본인 부담
4. 갱신 (Renewal) - 재가입 시 보험료 재산정
5. 면책사항 (Exclusion) - 미보장 항목
6. 청구 (Claim) - 보험금 요청
7. 기왕증 (Pre-existing Condition)
8. 갱신한도 (Max Age)

🐕 견종별 가이드 5개:
├─ 말티즈: 슬개골탈구, 디스크 → 기본형 추천
├─ 푸들: 귀염증, 피부질환 → 균형형 추천
├─ 비글: 이염증, 갑상선 → 보장형 추천
├─ 시바견: 알레르기, 슬개골 → 균형형 추천
└─ 고양이: 신장질환, 당뇨 → 예방적 가입 권장

📋 청구 절차 5단계:
1. 진료 후 서류 수집
2. 고객센터에 연락
3. 청구서 작성 및 제출
4. 심사 및 검증 (5-10일)
5. 보험금 지급

❓ FAQ 6개:
├─ 대기 기간은 몇 일?
├─ 기왕증도 보장?
├─ 청구 거절 이유?
├─ 보험료는 계속 올라감?
├─ 여러 회사 동시 가입 가능?
└─ 온라인 청구 가능?
```

## 📌 중요 참고사항

### 기술 스택
- **Frontend**: React 18.2.0 + Vite 5.0.8
- **Styling**: Tailwind CSS 3.4.0
- **AI**: Claude API (claude-opus-4-1-20250805)
- **Maps**: Google Maps API (Places + Markers)
- **Email**: Resend API
- **Database**: Supabase
- **Charts**: Recharts 3.7.0

### 배포 준비
- **Pre-deployment**:
  ```bash
  npm install
  npm run build
  npm run preview
  ```

- **Vercel 배포**:
  - 환경변수 6개 설정
  - 자동 배포 설정
  - 도메인 연결

### 성능 최적화
- Code splitting (컴포넌트별)
- Lazy loading
- Image optimization (서빙 필요시)
- API 캐싱
- LocalStorage 활용
